# ViVi PARK BD Blueprint

## 路徑

Cold mail
→ 表單 / 電話追蹤
→ 30 分鐘會議
→ 判斷合作路徑
→ 小規模驗證
→ 數據回顧
→ 雙方共同決定是否擴大、調整或停止

## 會議決策樹

| 問題 | 若答案是肯定 | 下一步 |
|---|---|---|
| ViVi 是否願意評估家庭能源服務進入生活服務？ | 是 | 討論服務券 / 首購券 / 會員導流 |
| Vi 幣或票券機制是否可承接旺來服務？ | 是 | 確認清算、折抵、退費、客服責任 |
| 場站是否適合作為線下履約節點？ | 是 | 桃園 / 台中 1 至 3 場現勘 |
| 權責或法規較複雜？ | 是 | 先走會員導流或商品券，不急著設櫃 |

## 反對意見回應

| 反對意見 | 回應 |
|---|---|
| 這是設備進駐嗎？ | 不完全是。設備只是其中一種履約載體，本案也可從會員導流或商品券先測試。 |
| ViVi 已經有生活服務合作 | 正因為 ViVi 已有生活服務基礎，旺來才適合提供家庭能源這種高頻、剛性、可回購的日常需求場景。 |
| 場站權責可能複雜 | 可以先不指定場站，先討論會員 / 點數 / 商品券路徑；若場站條件合適，再做 1 至 3 場試點。 |
| 會不會影響停車營運？ | 場站路徑只評估不影響收益、車位與動線的低干擾空間，並由旺來負責客服、補貨、維運、保險與異常處理。 |
