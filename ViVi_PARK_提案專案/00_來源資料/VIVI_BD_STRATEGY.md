# ViVi PARK BD 開發策略稿

> 2026-06-04 策略重估版：本文件已依據 `風格/公司策略介紹.pptx` 重新校準。  
> 重點修正：ViVi PARK 不應只被視為「停車場場地」或「Vi 幣平台」，而應被放進旺來 2026 三大策略中評估：零售通路結盟、瓦斯同業加盟、智能自取系統。

## 旺來策略對齊

公司策略簡報揭示旺來 2026 推進策略為：

1. **零售通路結盟策略**：上架會員招募、平台行銷導流、場地智取合作。
2. **瓦斯同業加盟策略**：供應商合作、夥伴轉加盟。
3. **智能自取系統策略**：作為 OMO 供應平台載體，也是快速複製與擴張關鍵。

因此 ViVi PARK 專案的核心問題不是「能不能放設備」，而是：

```text
ViVi PARK 是否能成為旺來 OMO 能源服務，在停車、點數、會員與零售生活場景中的驗證通路？
```

## 競合判斷

初步判斷：**不是直接衝突，屬於平台相似、品類互補的競合關係。**

ViVi PARK 做的是：

- 停車場經營與場站網絡。
- Vi 幣點數與會員帳戶。
- 停車、路停、加油、充電、生活服務的支付與票券場景。
- 與零售、點數、支付平台串接。

旺來做的是：

- LPG 家庭能源剛性需求。
- 智能自取櫃與 OMO 供應平台。
- 零售通路結盟與會員招募。
- 首配、安檢、配送、客服與異常處理。

兩者相似之處在於都在做「線上會員平台 + 線下服務節點 + 點數 / 金流 / 場域合作」。  
差異在於 ViVi PARK 的底層場景是車主與停車，旺來的底層場景是家庭能源與瓦斯回購。

## 一句話策略

把旺來瓦斯包裝成 ViVi PARK 可驗證的「家庭能源生活服務夥伴」：不是單純場地進駐，而是讓 ViVi 的停車會員、Vi 幣與生活服務平台，新增一個高頻且剛性的家庭能源使用場景。

## 合作定位

```text
ViVi PARK × 旺來瓦斯
家庭能源生活服務節點試點合作
```

## 為什麼找 ViVi PARK

1. ViVi PARK 已經不是單純停車品牌，而是 APP、Vi 幣、生活服務、點數漫遊平台。
2. ViVi PARK 的平台邏輯與旺來 OMO 策略相近，雙方可討論「會員導流 + 票券 / 點數 + 線下服務節點」的合作模式。
3. 停車場本身有社區、通勤、採買與生活圈接觸點，適合做 90 天小規模驗證。
4. ViVi PARK 需要持續擴大 Vi 幣與生活服務使用場景，旺來可提供 LPG 家庭能源這種高頻回購、剛性需求服務。
5. 若能建立合作範例，ViVi PARK 可取得停車以外的家庭生活服務案例；旺來可取得零售通路結盟與智能自取系統的停車場驗證場。

## 金流與通路關鍵觀察

公開資訊顯示：

- ViVi PARK 會員條款中，儲值點數可透過信用卡刷卡、ATM 匯款或購買頁面顯示之付款方式購買。
- ViVi PARK 官網最新消息列出「新增付款方式 - 富邦金流」。
- ViVi PARK 已上架 LINE Pay 購好券，使用者可在 LINE Pay App 內購買 Vi 幣點數。
- ViVi PARK 官方關於我們頁面揭露，2020 年曾獲台灣最大超市「全聯」委託經營停車場；官方案場資料亦有多個全聯場站。

對旺來的含意：

```text
ViVi 具備支付 / 點數 / 票券 / 零售合作經驗，
因此合作切入不應只談場地，而應同步評估：
1. Vi 幣 / 商品券是否能承接旺來瓦斯服務；
2. 智取櫃是否能作為線下履約節點；
3. 全聯等零售場站是否牽涉第三方權責，需謹慎處理。
```

## 建議切入順序

1. Cold mail 至客服信箱與官網表單，主旨鎖定「家庭能源生活服務合作」而非「租用場地」。
2. 附 OnePager DM，讓窗口理解本案同時具備會員導流、點數 / 商品券、場站服務節點三種價值。
3. 會議中先確認應走哪一條合作路徑：生活服務 / 特約商店、Vi 幣 / 商品券、或場站試點。
4. 若進入場站試點，再取得 1 至 3 個桃園／台中場站現勘或資料確認。
5. 90 天後以數據回顧：使用量、客訴、補貨、動線、會員轉換、Vi 幣 / 商品券整合可能。

## 不適合一開始談

- 全台展點。
- 要求 ViVi PARK 直接上架或直接分潤。
- 場地租金細節。
- SOP 與大量設備規格。
- 未確認權責前指定商場或全聯場站一定可做。

## 後續文件修正要求

後續 OnePager、Coldmail、PPT 應同步調整：

- 主軸從「停車後生活服務」升級為「家庭能源生活服務」。
- 不要只講場站邊角空間，要講「會員、點數、票券、線下履約」。
- 把 ViVi 視為可能的合作平台，不要預設為競爭者。
- 對全聯場站保持謹慎：可列為 ViVi 零售通路能力證據，但不應直接指定為試點，除非權責確認。
