# ViVi PARK 合作開發 DD 執行指南

> 2026-06-04 重估版：已納入旺來 2026 公司策略。後續所有 OnePager、Coldmail、PPT 不得只以「停車場場站」為主軸，必須同時評估平台、金流 / 點數、智能自取節點三條合作路徑。

## Project Objective

本專案目標不是取得單一停車場空間，而是建立旺來瓦斯與 ViVi PARK／榮帝科技的總部合作模式。

核心假設：

```text
ViVi PARK = 停車場網絡 + Vi 幣點數 + 生活服務平台
旺來 = LPG 家庭能源 + 智能自取系統 + 零售通路結盟 + OMO 供應平台
合作主軸 = 把旺來家庭能源服務接入 ViVi PARK 的會員、點數、票券與場站生活服務生態圈
```

## 旺來策略前提

執行本專案前，AI 必須理解旺來 2026 三大策略：

1. 零售通路結盟：上架會員招募、平台行銷導流、場地智取合作。
2. 瓦斯同業加盟：供應商合作、夥伴轉加盟。
3. 智能自取系統：OMO 供應平台載體、快速複製與擴張關鍵。

因此 ViVi PARK 的合作價值不是「停車場空間」，而是：

```text
停車場會員流量 + Vi 幣點數 / 商品券 + 零售場站 + 智能自取節點驗證
```

## DD Scope

本次採商業合作 DD，重點為：

1. 公司與品牌定位
2. 場站與聯盟模式
3. Vi 幣、生活服務、點數漫遊與會員生態
4. 金流 / 點數 / 商品券 / 票券合作能力
5. 零售通路合作與全聯場站權責
6. 桃園／台中可試點案場方向
7. 總部合作切入路徑

## 三條合作路徑

後續文件必須先判斷 ViVi 對哪條路徑有興趣：

| 路徑 | 內容 | 優先問題 |
|---|---|---|
| A. 生活服務 / 特約商店 | 旺來服務券、首購券、會員導流，上架 ViVi 生活服務 | 商品券、導流、客服與履約 |
| B. 金流 / 點數 | Vi 幣、富邦金流、LINE Pay 購好券、Hami Point / momo 幣等串接可能 | 清算、折抵、退費、帳務責任 |
| C. 場站 / 智取節點 | 1 至 3 個桃園 / 台中場站驗證智能自取或服務節點 | 供電、動線、保險、權責、現勘 |

會議前不應預設只走 C 路徑。

## 禁止說法

- 租停車格
- 廣告進駐
- 傳統設備投放
- 一開始大量展點
- ViVi 與旺來存在直接生態衝突
- 全聯場站一定可以由 ViVi 直接決定

## 正確說法

- 家庭能源生活服務節點
- ViVi 共榮生態圈的家庭能源服務延伸
- 點數 / 商品券 / 票券合作可行性
- 智能自取系統的低干擾試點
- 1 至 3 個示範場站，90 天驗證
- 雙方共同決定是否擴大、調整或停止
