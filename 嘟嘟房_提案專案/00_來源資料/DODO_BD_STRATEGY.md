# 嘟嘟房 BD 開發策略稿

> 目的：定義旺來瓦斯對嘟嘟房的開發定位、切入角度與推進策略。
> 本文件不對外使用，僅供旺來內部 BD 作業參考。

---

## 開發目標

取得嘟嘟房（寶盛國際）總部同意，進入 1 至 3 個試點場站，建立「複合型服務節點」合作模式。

---

## 為什麼找嘟嘟房

### 核心理由

嘟嘟房是台灣停車場品牌認知度第一、場站規模第二的連鎖停車品牌。

雖然已被城市車旅在規模上超越，但品牌力仍是所有停車品牌中最強的。

「嘟嘟房」幾乎是台灣消費者對「停車場」的代名詞。

---

### 嘟嘟房現在需要什麼

1. **場站差異化**：失去第一後，需要讓場站在「停車之外」提供更多價值
2. **IPO 成長故事**：寶盛國際計畫 2026～2027 上櫃，需要能展示成長性的新業務
3. **ESG 形象**：母公司中興電工積極布局能源事業，停車場端需跟上集團方向
4. **品牌翻新機會**：舊有停車場印象需要用創新服務更新

## 旺來對嘟嘟房的合作定位

### 對外說法（不能說）

```text
❌ 旺來想在嘟嘟房停車場放智取櫃
❌ 旺來想租嘟嘟房的停車格位
❌ 旺來想做廣告進駐
```

### 對外說法（應該說）

```text
✅ 旺來協助嘟嘟房場站建立複合型智慧服務節點
✅ 利用場站邊角空間，創造停車之外的生活服務價值
✅ 試點規模小、風險可控、不干擾現有停車業務
✅ 以 1 至 3 個示範場站驗證，雙方共同評估是否擴大
```

---

## 合作價值主張（對嘟嘟房的好處）

### 場站價值

活化停車場內閒置邊角空間，讓原本無收益的角落產生新的場域價值。

### 品牌價值

嘟嘟房場站形象從「傳統停車場」升級為「具備生活服務功能的智慧場域」，有助上櫃故事說服投資人。

### ESG 價值

結合中興電工集團能源背景，複合型服務節點（智取櫃 + 能源服務）強化集團 ESG 敘事。

### 會員價值

旺來服務可與嘟嘟房 APP 會員體系交叉推廣，提高使用者黏著度。

### 學習價值

透過 90 天試點數據，嘟嘟房可取得場站消費行為數據，作為後續場站規劃的決策依據。

```text
✅ 複合型服務節點設備
✅ 設備安裝與現勘評估
✅ 客服與消費者問題處理
✅ 補貨與庫存管理
✅ 設備維運與異常排除

```

---

## 嘟嘟房需要協助的事項（低負擔）

```text
✅ 提供場站基本條件資訊（面積、動線、供電）
✅ 安排現勘窗口
✅ 確認場站管理層配合
✅ 共同討論試點選址與評估標準
```

## 不適合一開始談的內容

| 主題 | 原因 |
|------|------|
| 點位數量超過 3 個 | 初期請求太大，容易讓對方望而卻步 |
| 合作費用與分潤模式 | 在未建立共識前談錢，容易引發警戒 |
| 場站設備規格細節 | 過早進入技術層，讓合作像設備推銷 |
| SOP 與作業流程 | 讓對方誤以為需要配合大量作業負擔 |
| 要求長期合約 | 試點階段不應有長期承諾壓力 |

## 1 至 3 個試點合作假設

### 試點場站條件（理想）

- 台北市核心商業區（信義、中山、大安）
- 場站邊角有 2 至 4 平方米可用空間
- 有供電條件（110V 或 220V 插座可用）
- 非轉租型場站（嘟嘟房直接管理）
- 每日進出車次 100 台以上

### 試點驗證指標（90 天）

| 指標 | 說明 |
|------|------|
| 使用次數 | 服務節點每日平均被使用次數 |
| 消費金額 | 平均消費單價與總金額 |
| 滿意度 | 使用者回饋評分 |
| 場站動線影響 | 是否影響停車業務動線 |
| 補貨頻率 | 服務穩定性評估 |
