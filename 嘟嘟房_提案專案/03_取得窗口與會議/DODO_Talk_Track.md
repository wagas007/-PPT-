## 開場（2 分鐘）

```text
非常感謝您撥出時間。我知道大家都很忙，所以我盡量 30 分鐘內說完。

旺來今天來，不是要談租格位，也不是廣告進駐。

我們只想提一個很小的嘗試：
讓嘟嘟房的場站，在消費者停好車之後，還有一件事可以順手做。

就這樣。
```

## 為什麼找嘟嘟房（4 分鐘）

```text
我們在評估合作對象時，嘟嘟房是第一順位。

理由很簡單：嘟嘟房是台灣停車場的品牌代名詞。
30 年、174 個場站。在台灣消費者的認知裡，「停車場」就是嘟嘟房。

這個品牌認知，是我們在任何其他停車品牌都找不到的。
如果旺來的服務能在嘟嘟房落地，曝光效益遠超過其他選項。
```

---

## 合作構想說明（8 分鐘）

```text
我們想做的事很單純：

  停車場入口旁，或電梯旁的邊角空間——
  通常是 2 至 4 平方米的閒置角落——
  放一個旺來的複合型智慧服務節點。

不佔任何停車格，不影響車道動線，設備整潔低調。

消費者停好車，可以順手做一件生活的事。就這樣。

旺來負責所有重要的事：
  設備、安裝、客服、補貨、維修、法規申請、責任保險。

嘟嘟房不需要配合任何日常作業。
我們唯一需要的，是場站的一個聯絡窗口，以及現勘的機會。
```

## 價值說明（4 分鐘）

```text
對嘟嘟房而言，這個試點有幾個潛在價值：

第一：閒置空間的活化。
  那個邊角，現在是什麼都沒有的空間。有節點之後，它有了場域功能。

第二：場站形象的升級。
  嘟嘟房的場站，從「停車場」多了一個「生活服務」的標籤。

第三：90 天的數據。
  試點結束後，旺來提供消費者行為報告。
  這份報告對嘟嘟房的場站規劃有參考價值，而且是零成本取得。

第四：（如對方有提到 IPO 或成長故事，再說）
  這個合作案，有機會成為場站加值的具體案例。
```

## 試點方式（4 分鐘）

```text
我們的建議：從桃園或台中選 1 至 3 個場站開始。

為什麼？旺來目前的拓展重心在桃園與台中。
先在旺來有資源配合的城市做試點，對雙方效率最高。

流程很簡單：
  Step 1：雙方確認場站條件（我們填一份簡單的問卷，30 分鐘現勘）
  Step 2：90 天試點，旺來全程主導
  Step 3：試點結束，雙方共同看數據，再決定下一步

90 天後，若數據不好看，我們收起設備，嘟嘟房沒有任何損失。
若數據好看，再一起討論下一步。
```

## 目標提問（2 分鐘）

> 關鍵：取得「可以讓我們去看幾個場站嗎？」的口頭同意。

```text
我的問題很簡單：

嘟嘟房在桃園或台中，有沒有幾個場站，
入口旁或電梯旁有一點點空間——
讓我們去看一下，確認現勘條件？

不用現在決定合作，就只是讓我們去看看。
```

## 常見問題應答

| 問題 | 回答要點 |
|------|---------|
| 你們要的是廣告位嗎？ | 不是廣告。是服務節點，消費者可以使用的設備，不是展示牌。 |
| 現在 IPO 期間，我們不談新合作。 | 了解。試點規模極小，設備由旺來提供，對財報無影響。可以先做非正式的場站條件確認嗎？ |
| 費用怎麼分？ | 這次來不是談錢的。先確認場站條件適不適合，再進入下一步討論。 |
| 你們想在台北做嗎？ | 旺來現在的重心在桃園與台中。我們想先從這兩個城市開始，效率最高。 |
| 需要總部審核才能決定。 | 完全理解。我們可以準備一份簡短的場站評估報告，幫您向總部說明。 |
