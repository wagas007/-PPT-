## 1. 嘟嘟房專案定位判斷

### 核心定位

**旺來 × 嘟嘟房 = 台灣品牌認知最強的停車場，建立複合型智慧服務節點。**

嘟嘟房（寶盛國際）是台灣停車場的品牌代名詞，174 個場站、台北市 84 站密集覆蓋。旺來找嘟嘟房，不是要租格位，而是借助其品牌黏著力與台北市核心場站網絡，在停車場的邊角建立複合型生活服務節點，讓停車客順手完成一件生活服務的事。

### 專案評分：A－

- **強烈建議投入**
- 主要限制：IPO 準備期（2026～2027 上櫃）審核趨嚴，需以低干擾、低承諾試點切入

### 三句話定位

```text
1. 嘟嘟房是台灣品牌認知最強的停車場，這個通路對旺來的曝光價值極高。
2. 失去市場第一後，嘟嘟房有場站加值的強烈動機，窗口期就是現在。
3. 複合型服務節點的概念與中興電工能源事業天然契合，結盟邏輯紮實。
```

## 2. 為什麼找嘟嘟房

### 主要理由（對外）

| 理由 | 說明 |
|------|------|
| 品牌認知第一 | 「嘟嘟房」在台灣幾乎是停車場代名詞，任何服務在嘟嘟房落地，曝光效益最大化 |
| 台北市密集場站 | 84 個台北市場站覆蓋核心商圈，試點選擇彈性最大 |
| 場站加值動機高 | 城市車旅 2022 年超越後，嘟嘟房有明確動機讓場站在停車之外提供更多價值 |
| IPO 成長故事 | 2026～2027 上櫃計畫，需要能展示場站加值、ESG、創新服務的新業務亮點 |
| 能源集團契合 | 母公司中興電工佈局太陽光電、氫燃料電池，複合型能源服務節點與集團方向完全吻合 |

### 現在是好時機的理由

1. 嘟嘟房失去龍頭地位，正在尋找場站差異化的答案
2. IPO 前的 1 至 2 年是「塑造成長故事」的關鍵期，新合作案有其政治動機
3. 嘟嘟房數位化已具備（APP、多元支付、HAPPY GO），服務整合門檻低
4. 城市車旅已在積極拓展異業合作，嘟嘟房若不動，差距只會拉大

## 3. 對嘟嘟房的合作價值

### 場站價值

停車場入口旁、電梯旁的邊角空間，原本無收益。複合型服務節點讓這些空間產生新的場域功能，活化閒置空間，不佔用任何停車格。

### 品牌價值

嘟嘟房場站形象從「傳統停車場」升級為「具備生活服務功能的智慧場域」。這是可放入上市說明書的創新合作案例，強化寶盛國際的成長敘事與投資人信心。

### ESG 價值

結合中興電工集團能源背景（太陽光電、氫能），複合型服務節點強化集團整體 ESG 形象，協助停車場端的 ESG 敘事落地，讓「能源集團」的故事在每個場站都能被看見。

### 會員與數據價值

- 嘟嘟房 APP 會員與旺來服務的交叉推廣機會
- 停車客行為數據可反饋給嘟嘟房，協助後續場站規劃決策
- 90 天試點數據報告由旺來提供，嘟嘟房零成本取得消費洞察

### 學習價值

嘟嘟房無需投入任何人力或資本，可透過試點觀察消費者對「停車場內生活服務」的真實接受度，為後續場站升級策略取得第一手數據。

## 4. 旺來可提供的能力

| 能力 | 說明 |
|------|------|
| 複合型服務節點設備 | 旺來提供設備，不需嘟嘟房投入任何硬體成本 |
| 設備安裝與現勘評估 | 旺來主導現勘與安裝，嘟嘟房只需提供場站窗口 |
| 客服與消費者問題處理 | 全部由旺來負責，不增加嘟嘟房客服負擔 |
| 補貨與庫存管理 | 旺來自主管理，嘟嘟房無需配合補貨作業 |
| 設備維運與異常排除 | 旺來負責所有設備維護與異常處理 |
| 法規申請與保險 | 旺來負責設備相關法規申請與責任保險 |
| 90 天試點數據報告 | 旺來提供完整試點分析報告，供嘟嘟房場站決策使用 |

**對嘟嘟房而言：一切重要的事都由旺來負責。**

## 5. 嘟嘟房需要協助的事項

刻意設計為「極低負擔」，降低對方決策阻力：

```text
1. 提供場站基本條件資訊（面積、動線、供電狀況）
2. 安排現勘窗口（確認場站管理人員可接待）
3. 確認場站管理層基本配合意願
4. 共同討論試點場站與驗證標準（一次 30 分鐘會議）
```

**嘟嘟房唯一真正要做的事：告訴我哪幾個場站的邊角條件可以評估一下。**

## 6. 一至三個試點合作假設

### 試點框架

```text
第一階段：1 至 3 個台北市試點場站
↓
90 天驗證（旺來負責設備、維運、數據）
↓
雙方共同評估結果，決定是否擴點
```

### 試點場站篩選條件

| 條件 | 說明 |
|------|------|
| 地理位置 | **桃園市或台中市**（旺來目前主要拓展城市）|
| 場站類型 | 嘟嘟房直接管理（非轉租型）|
| 空間條件 | 入口旁或電梯旁有 2 至 4 平方米邊角空間 |
| 供電條件 | 有 110V 或 220V 插座可用 |
| 流量門檻 | 日均進出車次 100 台以上 |

> ⚠️ 嘟嘟房在台北市場站最密集（84 站），但旺來目前拓展重心為桃園與台中。
> 試點選址應優先確認嘟嘟房在這兩個城市的直接管理場站數量與條件，再與對方共同討論。

### 推薦優先順序

| 優先級 | 場站城市 | 選擇理由 |
|--------|---------|---------|
| ★★★ | 桃園市 | 旺來主要拓展城市，嘟嘟房有場站分布，試點策略一致 |
| ★★★ | 台中市 | 旺來主要拓展城市，商業區停車需求強 |
| ★ | 其他城市 | 視嘟嘟房場站分布與雙方討論結果決定，台北備選 |

### 90 天驗證 KPI

| 指標 | 說明 |
|------|------|
| 日均使用次數 | 服務節點每日被使用的次數（目標：15 次以上）|
| 平均消費單價 | 每次使用的消費金額 |
| 停車業務干擾度 | 是否影響停車動線（目標：零投訴）|
| 消費者滿意度 | 使用者回饋評分 |
| 補貨頻率 | 服務穩定性評估 |

## 7. 不適合一開始談的內容

### 話題禁區

| 禁止主題 | 原因 |
|---------|------|
| 超過 3 個點位 | 初期請求太大，容易讓對方望而卻步 |
| 合作費用與分潤比例 | 在未建立共識前談錢，容易引發警戒 |
| 設備規格細節 | 過早進入技術層，讓合作像設備推銷 |
| 長期框架合約 | 試點階段不應有長期承諾壓力 |
| IPO 合作（主動提）| 讓對方自己說，不要搶先，避免引發敏感 |
| 廣告或媒體進駐 | 嘟嘟房對廣告進駐保持距離，會被歸類為騷擾 |
| SOP 與配合作業細節 | 讓對方誤以為需要配合大量作業負擔 |

### 第一句話定位（最重要）

任何接觸的第一句話，必須主動排除「租格位」誤解：

```text
「旺來這次找嘟嘟房，不是來租停車格，
 也不是廣告進駐，
 而是想跟嘟嘟房合作，
 在場站內建立一個複合型智慧服務節點。
 規模很小，風險我們扛，不影響你們停車業務。」
```

### 第一至二週執行（P1）

| 任務 | 目標 |
|------|------|
| 寄送 Cold mail | 附上 OnePager DM，切入「複合型服務節點」 |
| 電話追蹤確認收件 | 取得初步反應 |
| 安排 30 分鐘會議 | 確認場站條件與合作可行性 |

### 注意事項

- **IPO 時間敏感**：2026～2027 上櫃，2026 年下半年開始審核可能更嚴格，建議在 2026 年上半年完成第一次接觸
- **Cold mail 語氣**：商務簡潔，不強推，強調低干擾與旺來負責所有重要事項
- **第一次會議目標**：不談合約，只取得「允許現勘」的同意
- **現勘選站**：優先選擇信義、中山、大安的嘟嘟房直接管理場站
