# PARK24（Times）合作開發專案執行指南

## Project Objective

本專案目標並非取得單一停車場場地。

本專案目標為：

建立旺來瓦斯與 PARK24（Times）之總部合作模式。

透過總部合作：

* 建立試點合作
* 建立標準化模型
* 建立複製機制
* 建立台中與桃園後續擴點基礎

---

# 6. 合作適配性分析

## 6.1 合作交集分析

請驗證以下假設：

| PARK24需求 | 旺來可提供           |
| -------- | --------------- |
| 場站活化     | 利用閒置角落建立智慧服務據點  |
| 土地坪效     | 場域合作收益          |
| 低管理負擔    | 旺來負責設備、客服、補貨、維運 |
| 服務升級     | 社區服務節點          |
| 數據化管理    | 使用數據與會員數據       |
| ESG與品牌形象 | 智慧城市、社區服務       |

請補充：

* 是否存在其他合作價值
* 是否存在潛在衝突
* 是否存在品牌風險

---

## 6.2 合作定位驗證

禁止使用：

「租停車場放瓦斯櫃」

作為合作定位。

請統一採用：

「社區型智慧能源補給服務據點」

作為合作定位。

請驗證：

此定位是否符合：

* PARK24品牌策略
* PARK24土地活用策略
* PARK24智慧服務策略
